Keyboard shortcuts

Views
1 Unread entries
2 Saved entries
3 All entries
4 Feeds
Entry page
k Next
m Mark/unmark as read
s Save/unsave
v View entry on original website

Note: in order to work certain shortcuts require to disable the browser pop-up blocker.

Import and export your subscriptions

Use the sweat command-line script to import your subscription in OPML/XML format:

$ python sweat.py import /path/to/subscriptions.xml

Also, there is a export command to save your subscriptions in OPML/XML format:

$ python sweat.py export /path/to/subscriptions.xml

Refresh feeds

$ python sweat.py refresh

You should use Cron or similar utilities to schedule feeds refresh on a regular basis.
